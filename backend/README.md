Vidflyy Backend
================

Setup
-----

1. Copy `.env.example` to `.env` and fill values:
   - `MONGODB_URI`
   - `JWT_SECRET`
   - SMTP creds for OTP emails or leave blank for console transport
2. Install deps: `npm install`
3. Start dev server: `npm run dev`

Endpoints
---------

- Swagger docs: `/api-docs`
- Health: `/health`
